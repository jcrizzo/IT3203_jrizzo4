Browser Architecture

Modern browsers are made up of several connected parts that work together to load and display websites. These components help make web pages interactive, secure, and consistent for users.

  • User Interface: The visible part of the browser, such as the address bar, tabs, and buttons.
  • Rendering Engine: Reads HTML and CSS and turns them into the page users see.
  • JavaScript Engine: Runs scripts and allows websites to respond to user actions.
  • Networking: Handles communication between the browser and web servers.
  • Storage and Security: Helps manage cookies, local storage, permissions, and browsing safety.

Modern browser design often uses separate processes for tabs and rendering tasks. This improves stability and security because one broken tab is less likely to crash the whole browser.
